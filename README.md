# Homework

## Запуск

### Клонирование проекта и попадание внутрь проекта
```
git clone https://github.com/klvkna/homework.git
cd homework 
```

### Установка зависимостей
```
py -m pip install snakemake
```

### Запуск проекта
```
py -m snakemake --cores 1
```

## Программа
Программа считает количество букв "a" в тексте и сохраняет результат в отдельный файл

### Ожидаемый результат
После выполнения следующей команды:
```
py -m snakemake --cores 1
```

В проекте создадутся файлы:
1. `input.txt` - файл с текстом
```
apple banana orange avocado
```
2. `a_count.txt` - файл, содержащий число 7, которое соответствует количеству
букв "a" в текстовом файле.
